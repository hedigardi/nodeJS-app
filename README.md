# Node.JS Application

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hedigardi/nodeJS-app.git
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm run node-1
   ```
   OR/AND
   ```bash
   npm run node-2
   ```
